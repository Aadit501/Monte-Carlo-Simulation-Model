
# Personal Finance Monte Carlo Simulation 
Welcome to the Personal Finance Monte Carlo Simulation model! This project is an Excel-based financial planning tool designed to simulate thousands of possible financial futures based on varying life events, expenses, and income levels in South Africa. 

## Project Structure

The workbook (`Monte_Carlo_sim.xlsx`) is divided into four main sheets:

1. **Welcome Sheet**: An introduction to the model, basic navigation, and a little bit of spreadsheet humor!
2. **Assumptions**: The data-driven foundation of the model. This sheet outlines low and high-bound estimates for various living expenses (e.g., housing, transport, schooling, childcare, taxes, and groceries). All baseline data is sourced from real-world South African economic metrics and pricing.
3. **Monte Carlo Sim**: The computational engine. This sheet runs 1,000 distinct financial scenarios randomizing the variables defined in the assumptions to calculate *Total Money Left* (Net Worth/Savings).
4. **Histogram**: Visualizes the data generated by the simulation. It displays the probability distribution of different financial outcomes, letting you see the most likely financial scenarios at a glance.
5. **Reflection**: A breakdown of the technical skills developed (Excel proficiency, problem-solving, data management) and personal financial takeaways (the realization of how expensive kids are, the compounding cost of coffee, and the importance of mindful spending!).

## Key Takeaways
Financial planning isn't just about hard numbers; it's about aligning goals and values with money. This project serves not just as a mathematical tool, but as a mirror showing how lifestyle choices (and skipping the occasional cup of coffee) can snowball into significant long-term financial shifts.
